# Mud Reports

## Overview

Full-stack application with React + TypeScript frontend and FastAPI + SQLite backend.

## Tech Stack

- **Frontend**: React 18, TypeScript, Vite, Axios, React Router
- **Backend**: FastAPI, SQLAlchemy, SQLite, Pydantic
- **Database**: SQLite (app.db)
